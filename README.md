## 项目技术栈

`语言`: typescript

`框架`: vue3

## 开发规范
### 分支规范
- 开发分支需以dev-开头 具体格式为dev-feature
- 所有变量满足驼峰式命名
- 代码合并需至少一人进行code review

项目开发
```bash
npm run dev | yarn dev | vite | npx vite
```

项目打包
```bash
npm run build | yarn build 
```
